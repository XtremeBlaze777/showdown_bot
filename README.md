Basically me attempting to vibe code a "Bot" mode for Pokemon Showdown.

Usage:
```bash
cd pokemon-showdown
npm install  # I like to do this in a nodeenv
node pokemon-showdown start --no-security &  # or just run it in a separate terminal if you want, i'm fine just hitting it with pkill
cd ..
# paste in the team you want the bot to use in main.py
py main.py
```
